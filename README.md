# Customer Revenue Prediction Engine

## Problem Statement
Businesses struggle to predict customer purchase behavior and future revenue, leading to poor targeting and lost revenue opportunities.

## Objective
Build an end-to-end machine learning system that:
- Predicts purchase probability
- Forecasts next-period revenue
- Identifies key revenue drivers

## Dataset
UCI Online Retail Dataset

## Tech Stack
- Python
- Pandas
- Scikit-learn
- XGBoost
- Streamlit
